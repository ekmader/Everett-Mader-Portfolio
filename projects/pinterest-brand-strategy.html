27-slide narrative deck, mobile-friendly, interactive browsing experience.

This project is built as an interactive slide deck experience designed to let a viewer “flip through” a presentation without downloading a file. It preserves layout, typography, and quick transitions while keeping navigation in the browser.

Now using native image-based slide navigation: each slide is loaded from img/Pinterest/Pinterest Project/1.png ... img/Pinterest/Pinterest Project/27.png. The deck is fluid and mobile-friendly.
